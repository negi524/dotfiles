# Hello World Skill

Claude Skillsの動作確認用のシンプルなスキルです。

## 機能

このスキルは以下の簡単なタスクを実行します：

- 挨拶メッセージの表示
- 現在の日時の表示
- 簡単なファイル情報の確認

## 使用例

スキルを実行すると、以下のような情報を提供します：

```
こんにちは！Claude Skillsが正常に動作しています。
現在時刻: 2024-XX-XX XX:XX:XX
作業ディレクトリ: /path/to/current/directory
```

## 目的

- Claude Skillsの基本動作確認
- 環境情報の簡単な取得
- スキル機能のテスト

動作確認用のため、複雑な処理は含まれていません。
