# 学習記録（Learnings）

## このディレクトリについて

pr-learningスキルで学習した内容を、言語・フレームワーク別に整理して保存します。
コードレビュースキル実行時に自動参照されます。

## ファイル構成

| ファイル | 対象 |
|---------|------|
| typescript.md | TypeScript固有のパターン |
| react.md | React（コア機能、Hooks等） |
| react-router.md | React Router |
| vue.md | Vue / Nuxt |
| rails.md | Ruby on Rails |
| python.md | Python |
| nestjs.md | NestJS |
| general.md | 言語非依存の汎用パターン |

## 記法

各ファイルは以下の形式で記述：

### 推奨パターン
- **パターン名**: 説明
- **理由**: なぜ推奨か
- **例**: コード例
- **学習元**: PR番号やプロジェクト名

### アンチパターン
- **パターン名**: 説明
- **問題**: 何が問題か
- **代替案**: どうすべきか
- **学習元**: PR番号やプロジェクト名

## 使い方

### 学習内容の追加

1. pr-learningスキルを実行
2. 学習レポートから重要なパターンを抽出
3. 該当する言語/フレームワークファイルに追記
4. 学習元（PR番号、日付）を明記

### レビュー時の参照

コードレビュースキル（code-review-bugs, code-review-simplify, code-review-maintainability）実行時に、
該当する言語/フレームワークの学習記録が自動参照されます。

- 推奨パターンに合致 → 称賛
- アンチパターンに該当 → 指摘・改善提案
- 新しいパターン発見 → 学習記録への追加を提案
